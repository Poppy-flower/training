11-第十一章 表单

一、form 表单

1.<form action='...../xxx.asp' method='get' name='' ></form>

表单属性

action 规定当提交表单时向何处发送表单数据
method 规定用于发送 form-data 的 HTTP 方法

  • get
  • post

target 规定action 属性中提交的页面会在何处打开

  • _blank
  • _self

name 表单名称

二、form中的input控件

1.<input type="…… " name="" value="" />

input中的 type类型

  • text 文本框
  • password 密码
  • radio 单选 要指定相同的name
  • checkbox 复选
  • label 标签为 input 元素定义标注

  • <input type="radio" name="" id="a" />
    <label for="a">……</label> for 属性应当与相关元素的 id

  • checked 属性定义默认选中的 input 元素。
    <input type="checkbox" name="" checked/>

  • disabled 属性规定应该禁用 input 元素。
    <input type="checkbox" name="" disabled/>
  • submit 提交form中的数据
  • reset 重置
  • button 按钮 ,多数情况下，用于通过 JavaScript 启动脚本
  • image 图片（有提交功能）
  • file 上传，供文件上传
  • hidden 隐藏

三、form中的select/option控件

select/option 下拉选框

<select size="2">
<option value='2017'>2017</option>
<option value='2017'>2018</option>
<option value='2017'>2019</option>
</select>
size 规定下拉列表中可见选项的数目
<option selected='selected'></option>
selected 规定在select里面默认展示第几项

高度的支持不兼容

四、form中的textarea 控件

textarea 文本域
属性:

rows 属性定义 高度
cols 属性定义 宽度
resize 调整尺寸样式属性（css3）

  • none 禁止调整
  • vertical 可调整高度
  • horizontal 可调整宽度

五、form中的fieldset分组

<fieldset> 可将表单内的相关元素分组
<legend> 标签为 fieldset 元素定义标题。

六、焦点focus

focus 伪类，获取焦点,适用a,input,button

1.input:focus{}

outline 外轮廓线,适用a,input,button

1.input,a,button{

2.        outline:1px solid red;

3.}

placeholder属性 定义提示信息（ccs3）

1.<input type="text" name="" id="a"  placeholder='请输入密码'/>
