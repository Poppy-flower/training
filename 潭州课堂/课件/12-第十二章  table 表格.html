12-第十二章 table 表格

一、table表格结构

table 定义标签表格
thead 定义标签表格头

  • tr 定义表格行
  • th 元素定义表头单元格

tbody 定义表格主体

  • tr 定义表格行
  • td 定义内容单元格

tfoot 定义表格尾

  • tr 定义表格行
  • td 定义内容单元格

二、table属性

属性：

  • width 规定表格的宽度。
  • height 规定表格的高度。
  • border 规定表格边框的宽度 border=’1’

样式属性 ：

  • border-spacing:X Y 指定单元格边界之间的水平和垂直间距
  • border-collapse:collapse 如果可能，边框会合并为一个单一的边框。会忽略 border-spacing

二、table样式重置

1.table{border-collapse:collapse;}

三、 table的默认特性

1、单元格默认平分table 的宽度
2、th里面的内容默认加粗并且左右上下居中
3、td里面的内容默认上下居中 左对齐显示
4、 table 决定了整个表格的宽度；
5、table里面的单元格宽度会被转换成百分比；
6、表格同一列/行会继承最大值；
7、th,td中没有margin

=>单元格可以包含:(文本、图片、列表、段落、表单、表格) 等等

四、 table的单元格合并

rowspan 属性规定单元格跨的行数 

1.<td rowspan="2"></td>

colspan 属性规定单元格跨的列数 

1.<td colspan="2"></td>

五、 display:table的特征

  • 支持margin:auto;
  • 默认内容撑开宽度；
  • 支持宽高
  • 表格前后带有换行符。
