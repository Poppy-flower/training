1.介绍一下标准的CSS盒模型？它和低版本的IE盒模型有什么区别？ （1）有两种：IE盒模型、W3C标准盒模型 （2）盒模型：内容（content）、填充（padding）、边框（border）、边界（margin） （3）区别：IE的content部分把padding和border算了进去。 CSS3里面有什么语法能控制浏览器以什么方式解析盒模型？ box-sizing的语法： box-sizing:content-box||border-box; 其中：content-box，告知浏览器以标准模式解析，也是默认模式； border-box，告知浏览器以怪异模式解析； 介绍一下CSS的盒子模型？ 1）有两种， IE 盒子模型、标准 W3C 盒子模型；IE的content部分包含了 border 和 pading; 2）盒模型： 内容(content)、填充(padding)、边界(margin)、 边框(border). 4.IE8以下版本的浏览器中盒模型有什么不同？ IE8以下的版本浏览器的盒模型中定义的元素的宽高不包括内边距和边框。
